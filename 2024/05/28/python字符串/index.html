普通字符串（String）

  • 表示形式：用单引号 '...' 或双引号 "..." 包围。
  • 数据类型：str。
  • 用途：表示常规文本。

原始字符串（Raw String）

  • 表示形式：以 r 或 R 开头，用单引号或双引号包围，如 r'...' 或 r"..."。
  • 用途：用于处理包含反斜杠（\）的字符串，例如正则表达式、文件路径等。反斜杠不会被解释为转义字符。

示例：

字节字符串（Byte String）

  • 表示形式：以 b 或 B 开头，用单引号或双引号包围，如 b'...' 或 b"..."。
  • 用途：表示二进制数据或需要特定编码的文本。

示例：

格式化字符串（Formatted String Literal）

  • 表示形式：以 f 或 F 开头，用单引号或双引号包围，如 f'...' 或 f"..."。
  • 用途：用于插入变量或表达式的值，生成动态内容。

Unicode字符串（在Python 2中）

  • 表示形式：以 u 或 U 开头，用单引号或双引号包围，如 u'...' 或 u"..."。
  • 用途：表示Unicode字符串。Python 3中默认所有字符串都是Unicode字符串，所以不需要使用前缀。

多行字符串（Multiline String）

  • 表示形式：用三个单引号 '''...''' 或三个双引号 """...""" 包围。
  • 用途：表示跨多行的字符串。

总结

Python中提供了多种字符串类型来处理不同场景下的文本数据。选择合适的字符串类型可以提高代码的可读性和处理效率：

  • 普通字符串：用于常规文本。
  • 原始字符串：用于包含反斜杠的文本（如正则表达式、文件路径）。
  • 字节字符串：用于处理二进制数据。
  • 格式化字符串：用于插入变量或表达式的值。
  • 多行字符串：用于表示跨多行的文本。
  • 模板字符串：用于替换占位符。

根据具体需求选择合适的字符串类型，可以使代码更简洁和高效。
