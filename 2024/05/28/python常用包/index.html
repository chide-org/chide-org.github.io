python常用包

五月 28, 2024 9218

base

base：内置函数

numpy

numpy：主要用于高性能的数值计算和数组操作。

pandas

pandas：构建在 NumPy 之上，专注于数据操作和分析，尤其是表格数据。

scipy

scipy: 一个基于NumPy的Python库，提供了许多用于科学和工程计算的函数。它包含了线性代数、积分、优化、信号处理和统计等方面的工具。

Matplotlib

Matplotlib: 一个广泛使用的Python 2D绘图库，用于生成图表和可视化数据。

seaborn

seaborn:一个基于 Matplotlib 的 Python 数据可视化库，专注于统计可视化，与Pandas和NumPy等库无缝集成。

networkx

networkx：一个用于创建、操作和研究复杂网络结构的Python库。它提供了丰富的功能用于处理图（网络）数据结构。

os

os: 提供了许多用于操作文件和目录的函数，允许你与操作系统进行交互。

scikit-learn

scikit-learn是Python中最流行的机器学习库之一，提供了丰富的功能和工具来实现各种机器学习任务。
