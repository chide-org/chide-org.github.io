直接用下面的命令很大可能会报错

进入db-sample-schemas-21.1文件目录，执行perl命令

  • perl: 这指定了要运行的程序，即 Perl 解释器。
  • -p: 这个选项告诉 Perl 在处理输入时，打印每行的结果。
  • -i.bak: 这个选项告诉 Perl 在原始文件上进行 “in-place” 编辑，并将原始文件备份为扩展名为 .bak 的文件。
  • -e: 这个选项允许你在命令行上提供 Perl 代码。
  • 's#__SUB__CWD__#'$(pwd)'#g': 这是 Perl 代码的一部分。它使用正则表达式将 __SUB__CWD__ 替换为当前工作目录的路径。$(pwd) 是一个 Shell 命令，用于获取当前工作目录的路径。
  • *.sql*/*.sql*/*.dat: 这是文件名的模式。它告诉 Perl 在当前目录下找到所有以 .sql 或 .dat 结尾的文件。
