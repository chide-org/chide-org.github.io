我是windows系统，之前尝试用 jekyll 搭建博客，发现在windows上会出现很多问题，比如界面css丢失，解决办法很繁琐，于是转到 hexo ，意外的发现 hexo 在windows上的搭建非常简单优雅，特意写了一个小结作为记录。

B站原教程传送门

# 环境配置

  • 下载node.js，官网链接，和node一起下载的还有包管理器npm（可以当作一个软件商店）

  • （可选）如果直接用npm进行hexo下载时出现问题，可能是网络问题，更换映射即可，自己搜索

# hexo框架下载

  • 下载hexo，hexo是一个静态网页生成器，并不是一个博客编辑器，hexo只负责 把md文件渲染成网页，如果想要更好的编辑体验，可以尝试Typora，购买可以自行搜索“数码荔枝”（中国大陆代理）

  • 创建并初始化博客文件夹，任意找一个位置，创建一个任意名称的文件夹，我命名为blog
